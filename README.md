# 🤖 Telegram AI Bot

A powerful Telegram bot powered by multiple AI models (ChatGPT-like). Users can select their preferred AI model and get instant responses.

## Features

✨ **Multi-AI Support**
- OpenAI GPT-4
- Google Gemini
- Anthropic Claude
- Easily add more models

🔐 **Channel Verification**
- Users must follow your channel before using the bot
- Automatic membership checking

⚙️ **Flexible Configuration**
- Add/remove AI providers with ease
- Extensible architecture

📝 **User Management**
- Track user interactions
- Per-user model selection
